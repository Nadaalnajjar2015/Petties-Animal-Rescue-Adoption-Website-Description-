## Description
Petties is a web application designed to help animal lovers and rescue groups post, find, and adopt animals safely. The platform provides an easy-to-use interface for managing animal rescue, adoption, lost pets, and mating posts.

## Features
- User Authentication: Sign in to post, search, and interact with listings
- Post Management: Create and view posts for rescued animals, animals available for adoption, lost pets, and mating opportunities
- Search & Filters: Search for animals based on type, location, age, or special requirements
- Interactive Gallery: View animals with full details including images, age, breed, and location
- Responsive Design: Soft colors and illustration-based interface

## Technologies
- HTML, CSS, JavaScript
- User-Centric UI design
- Responsive layouts for a friendly user experience
